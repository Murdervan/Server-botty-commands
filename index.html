Server Botty - Command Overview

All available commands grouped and sorted

📜 Info

  • !joinleaveinfo — Shows recent join/leave events from the log channel
  • !roleinfo [role] — Shows info about a role
  • !serverinfo — Shows info about the server
  • !userinfo [user] — Shows info about a user

⚙️ Moderation

  • !autorole add/remove [role] — Manage autoroles for new members
  • !ban @user [reason] — Ban a user
  • !clean [amount] — Clean messages in channel
  • !kick @user [reason] — Kick a user
  • !mute @user [duration] — Mute a user
  • !unmute @user — Unmute a user

💰 Economy

  • !addmoney @user [amount] — Add money to a user (admin only)
  • !bal [user] — Check balance

🎮 Games

  • !blackjack [bet] — Play blackjack
  • !coinflip [bet] — Coinflip game
  • !roll [sides] — Roll a dice
  • !roulette [bet] [red/black/green/number] — Play roulette
  • !rps [rock/paper/scissors] — Play Rock-Paper-Scissors

⏰ Miscellaneous

  • !invite_statbot — Get invite link for StatBot
  • !remind [time] [message] — Set a reminder
  • !help — Show this command list
