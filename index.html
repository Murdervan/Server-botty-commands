For help join: https://discord.gg/9NdbUZc8j5

🎲 HOW TO PLAY

  • Blackjack: react 👊 to Hit, ✋ to Stand, 💰 to Double Down, ✌ to Split.
  • Roulette: !roulette [bet] [red/black/green/number] — bet on a colour or specific number.
  • Coinflip: !coinflip [bet] — choose your side via reactions (heads/tails).

💰 Casino Games

  • !addmoney — Add coins to a user (admin only)
  • !bal — Check your balance or another user's balance
  • !blackjack — Play blackjack with betting. Usage: !blackjack [bet] (includes Split & Double Down)
  • !coinflip — Flip a coin with betting. Usage: !coinflip [bet] (choose via reactions)
  • !drop — Drop coins for members with the 'Casino' role (embed only)
  • !roll — Roll a die. Usage: !roll [sides]
  • !roulette — Play roulette with betting. Usage: !roulette [bet] [red/black/green/number]

📜 Info

  • !joinleaveinfo — Show join/leave statistics for the server
  • !roleinfo — Show info about a role. Usage: !roleinfo [role name]
  • !serverinfo — Show info about the server
  • !userinfo — Show info about a user. Usage: !userinfo @user

🎮 Games & Quiz (no betting)

  • !fakenews — True/False quiz. Answer with ✅ or ❌ (5 questions)
  • !flagquiz — Name the flag! 10 rounds
  • !memory — Emoji memory game (Simon). Usage: !memory
  • !reverse — Reverse your text. Usage: !reverse [text]

🔍 Search & Information

  • !gameinfo — Get Steam game information. Usage: !gameinfo [title]

😂 Fun & Social

  • !overdriv — Exaggerate a sentence. Usage: !overdriv [text]
  • !roast — Light roast (for fun). Use !roast @user or !roast to roast yourself
  • !systemmsg — Random system message

⏰ Misc

  • !helpadmin — Show admin-only commands
  • !helpme — Show this help menu with all commands (English)
  • !invite_statbot — Get the StatBot invite link
  • !remind — Set a reminder. Usage: !remind [time] [message] (e.g., !remind 5m Take a break)
  • !report — Report a user to the staff channel. Usage: !report @user [reason]

🤖 Automatic Log Actions (info)

  • Welcome messages
  • Birthday pings
  • Keyword triggers
  • Random emoji reactions
  • Laughs on 3+ 😂/🤣
  • Rare loot drops
  • 'Shhh mode' anti-spam

Birthday pings: Use !birthday set YYYY-MM-DD to register your birthday. The bot will ping you on that date (local server time) with a celebration message.
To list saved birthdays use !birthday list. Server staff can view the birthday list with !birthday list (requires Manage Messages permission).

Tip: consider muting the log channel if it gets too noisy.
