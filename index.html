🤖 Server Botty — Command Overview

All available commands grouped by category

🎲 HOW TO PLAY

  • Blackjack: react 👊 to hit, ✋ to stand, 💰 to doubledown, ✌ to split.
  • Roulette: !roulette [bet] [red/black/green/number]
  • Coinflip: !coinflip [bet], choose via reactions.

💰 Casino Games

  • !addmoney — Add coins to a user (admin only)
  • !bal — Check your balance or another user's balance
  • !blackjack — Play blackjack with betting. Usage: !blackjack [bet] - INCL SPLIT AND DOUBLEDOWN
  • !coinflip — Flip a coin with betting. Usage: !coinflip [bet]
  • !drop — Drop coins for members with the 'Casino' role (any Casino member can trigger)
  • !roll — Roll a die. Usage: !roll [sides]
  • !roulette — Play roulette with betting. Usage: !roulette [bet] [red/black/green/number]

📜 Info

  • !joinleaveinfo — Show join/leave statistics for the server
  • !roleinfo — Show info about a role. Usage: !roleinfo [role name]
  • !serverinfo — Show info about the server
  • !userinfo — Show info about a user. Usage: !userinfo @user

🎮 Games & Quiz (no betting)

  • !fakenews — True/False quiz. Answer with ✅ or ❌ (5 questions).
  • !flagquiz — Name the flag! 10 rounds.
  • !memory — Emoji memory game (Simon). Usage: !memory
  • !reverse — Reverse your text. Usage: !reverse [text]

🔍 Search & Information

  • !cur — Convert currency rates. Usage: !cur [amount] [FROM] [TO]
  • !gameinfo — Get Steam game information. Usage: !gameinfo [title]
  • !track — Track packages using tracking number. Usage: !track [tracking-number]

😂 Fun & Social

  • !burger — Builds a random burger
  • !overdriv — Overdriv en sætning. Usage: !overdriv [tekst]
  • !pause — Dramatic pause... then a punchline. Usage: !pause [message]
  • !roast — Light roast (for fun)
  • !systemmsg — Random system message

⏰ Misc

  • !helpadmin — Show admin-only commands
  • !helpme — Show this help menu with all commands (English)
  • !invite_statbot — Get the StatBot invite link
  • !remind — Set a reminder. Usage: !remind [time] [message] (e.g., !remind 5m Take a break)
  • !report — Report a user to the staff channel. Usage: !report @user [reason]
