Commands website: https://is.gd/Serverbotty_commands

💰 Casino Games

  • !bal — Check your balance or another user's balance
  • !drop — Random drops for members with the '@Casino' role — Max 3 per hour
  • !scoreboard — See who the big winner is
  • !blackjack 100 — Play blackjack with betting. React 👊 to Hit, ✋ to Stand, 💰 to Double Down, ✌ to Split
  • !roulette 50 red — Play roulette (red/black/green)
  • !coinflip 100 — Flip a coin with betting (choose via reactions)
  • !addmoney @user 1000 — Add coins to a user (admin only)

🎮 Games & Quiz (no betting)

  • !roll 6 — Roll a dice (1–6)
  • !fakenews — True/False quiz (✅ or ❌)
  • !flagquiz — Name the flag! 10 rounds
  • !memory — Emoji memory game (Simon)

😂 Fun & Social

  • !birthday 25-12-2000 — Register your birthday
  • !reverse hello world — Reverse your text
  • !overdriv this is amazing — Exaggerate a sentence
  • !roast — Light roast (for fun)
  • !systemmsg — Random system message

🔍 Search & Information

  • !gameinfo Minecraft — Get Steam game information
  • !invite statsbot — Get the Live Stats on your server

📜 Info

  • !remind 5m Take a break — Set a reminder
  • !joinleaveinfo — Show join/leave stats
  • !roleinfo Member — Show info about a role
  • !serverinfo — Show server info
  • !userinfo @user — Show info about a user

⏰ Misc

  • !report @user spam in chat — Report a user
  • !helpme — Show all commands
  • !helpadmin — Show admin-only commands

🤖 Automatic Log Actions (info)

Welcome messages • Birthday pings • Keyword triggers • Random emoji reactions • Laughs on 2+ 😂/🤣 • Rare loot drops • 'Shhh mode' anti-spam

Birthday pings: Use !birthday [DD-MM-YYYY] to register your birthday. The bot will ping you on that date (local server time) with a celebration message. Only server staff can view the full birthday list.
