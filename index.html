  • !drop — Random drops for members with the Casino role - Max 3 per time
  • !guessdrop 1-25 — Guess a number. Bot picks random number. Correct guess gives 2500-10000 coins - Max 3 per day
  • !scoreboard — Who is the big Winner?
  • !blackjack 100 — Play blackjack with betting. React 👊 Hit, ✋ Stand, 💰 Double Down, ✌ Split.
  • !roulette 50 red — Play roulette with betting. Choose red, black, green, or a number
  • !coinflip 100 — Flip a coin with betting. Choose via reactions red or blue

🛡️ Admin Commands

💰 Economy

  • !addmoney @user 1000 — Add coins to a user
  • !delmoney @user 500 — Remove coins from a user
  • !resetbal @user — Reset balance to 0 for a user or all Casino role members

🎂 Birthday Management

  • !birthday @user 25-12-2000 — Set a birthday for another user
  • !birthday list — List all saved birthdays
  • !birthday del @user — Delete a user's birthday

🔒 Channel Management

  • !lock #channel — Lock a channel so members cannot send messages
  • !unlock #channel — Unlock a channel
  • !say #channel your message — Bot sends message in chosen channel for announcements
  • !channel list — Show names of all text channels (easy to copy for !say or !lock)

⚠️ Moderation

  • !warning @user reason — Warn a user
  • !mute @user 10m — Mute a user (supports s, m, h, d)
  • !unmute @user — Unmute a user
  • !kick @user reason — Kick a user
  • !ban @user reason — Ban a user
  • !clean 50 — Clean messages (or use ALL to delete all)

👥 Server Management

  • !autorole add RoleName — Add an autorole given to new members
  • !autorole remove RoleName — Remove an autorole
  • !member list — Show server member list
  • !joinleaveinfo — Show join/leave statistics for the server
