Server Botty - Command Overview

All available commands grouped and sorted

🎯 HOW TO PLAY

  • Blackjack: type hit for a card, stand to stay.
  • Roulette: !roulette [bet] [red/black/green/number]
  • Coinflip: !coinflip [bet], choose heads/tails.
  • Roll: !roll [number of sides]

📜 Info

  • !joinleaveinfo — Shows recent join/leave events from the log channel
  • !roleinfo [role name] — Shows info about a role
  • !serverinfo — Shows info about the server
  • !userinfo [username] — Shows info about the user

⚙️ Moderation

  • !autorole add/remove — Manage autoroles (Subcommands: add, remove)
  • !ban @username [reason] — Ban a user
  • !clean [amount] — Clear messages in the channel
  • !kick @username [reason] — Kick a user
  • !mute @username 10m — Mute a user
  • !unmute @username — Unmute a user

💰 Economy

  • !addmoney — Add money to a user (admin only)
  • !bal — Check balance for yourself or another user

🎮 Games

  • !blackjack — Play blackjack with betting
  • !coinflip — Flip a coin with betting
  • !roll — Roll a die
  • !roulette — Play roulette with betting

⏰ Others

  • !invite_statbot — Get the StatBot invite link
  • !remind — Set a reminder. Usage: !remind 10m message
